## 🎨 UI Customization

1. **Centered or Left-aligned Start Menu?**
   - Settings → Personalization → Taskbar → Taskbar behaviors → Alignment.

2. **Per-app Dark/Light Mode.**
   - Now you can set "Dark" for Notepad and "Light" for File Explorer independently.

3. **Full-screen Widgets.**
   - Press `Win + W`, then click the icon in the top-right corner → "Expand".

4. **Custom Sound Schemes.**
   - Drop your own `.wav` files into `C:\Windows\Media` and select them in sound settings.

## 🔧 Terminal & Dev Tools

If you're a developer, these are for you:

- **Windows Terminal is now default**: Configure your profile with `"startingDirectory": "%USERPROFILE%/projects"`.
- **WSL 3 (announced 2026)**: Native GPU support without passthrough. Install with `wsl --install --version 3`.
- **Dev Drive**: Create a virtual ReFS drive for your projects using the `DevDrive` PowerShell command — compile times improve by up to 30%.
- **Git integration in File Explorer**: Right-click any folder → "Open in GitHub Desktop" or "Clone here".
- **Dev Home App**: The new dashboard shows CPU, memory, and network stats at a glance. Pin it to your taskbar.

## 🛡️ Security & Privacy

| Feature | How to enable |
|---------|---------------|
| **Smart App Control** | Settings → Privacy & Security → Smart App Control. Blocks untrusted apps automatically. |
| **Encrypted DNS** | Settings → Network & Internet → DNS → Enable "Encrypted DNS (DoH)". |
| **Local AI Processing** | In Privacy settings, disable "Send voice data to cloud" — all AI features will run locally. |
| **Privacy Dashboard** | Press `Win + I`, search "Privacy". See which apps accessed your camera/mic in the last 7 days. |

## 📊 System Optimization

- **Storage Sense 2.0**: Settings → System → Storage → Enable "Intelligent cleanup". It now deletes cached updates automatically.
- **Game Mode AI**: Turn on in Settings → Gaming → Game Mode. The system learns your gaming habits and prioritizes resources accordingly.
- **Startup Manager**: Press `Ctrl + Shift + Esc` → Startup → Enable "Show impact" column to see which apps slow your boot.
- **Power Plan**: Use `powercfg /setactive 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c` for ultimate performance (High Performance plan).
